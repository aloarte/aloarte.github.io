Featured projects

The Skintker project

This project involves an Android application and a backend service that exposes an API used by the application. Through these services, users can log their daily routines to track skin issues with the goal of identifying their root causes.

It utilizes Firebase for application authentication, where user credentials are used to bind logs in the backend, and the associated token is employed to secure API calls.

The technologies used are the following:

  • OAuth with Firebase
  • Kotlin
  • Koin
  • Jetpack-Compose in the Android application
  • Git+Github with Github Actions
